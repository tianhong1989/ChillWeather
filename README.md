# ChillWeather
轻量的Android天气查看应用，只为我的可爱制作。

## 特点
- 轻量，无多余功能
- 遵循Material Design界面设计规范
- 支持多城市管理
- 支持自定义背景图片
- 本天气App为私人定制，未曾想过发布，所以版本兼容性很差
- 本程序开源，遵守Apache 2.0协议
